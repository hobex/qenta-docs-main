getOrderDetails

getOrderDetails

This operation returns all details for a given order including all possible operations, corresponding payments and credit notes.

The required order of the parameter values when computing the fingerprint is: customerId, shopId, toolkitPassword, secret, command, language and orderNumber

Additional required request parameters

Parameter Data type Short description

orderNumber

numerical with up to 9 digits

order number of payment

Additional response parameters

Within the additional response parameters are some parameters which will occur more than once. Therefore we use the letters "n" and "m" as placeholders for digits. Currently "n" is always 1 because this operation returns only one order. The letter "m" indicates that there may be more than one payment or credit objects returned.
