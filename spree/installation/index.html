Wirecard Shop Plugin for Spree 2

Wirecard Shop Plugin for Spree 2

Installation guide

This installation guide will show you a step-by-step installation of the plugin to your installed shop system on your web server. Please test your web shop and the configuration of the plugin on a test system, before you install the plugin on your production system.

Step 1

Please log in to the administration area of your web shop.

Please select the "Plugin Manager" within the "Configuration".

Step 2

Please select the manual adding of a new plugin and choose the zip file provided by Wirecard CEE. After uploading the zip file you get the information of your web shop that the plugin has been successfully uploaded. The plugin will now be displayed within the list of "inactive Plugins".

Step 3

Please click on the green "(+)" icon to install the plugin to your web shop.

After successful installation you are able to configure the plugin, therefore please select "Configure plugin" within the dialog that occurs after the successful installation. A new window opens where you are able to configure the details of the plugin.

Configuration of the plugin

Within the "Detail site" of the plugin you are able to edit the configuration within the tab "Settings".

Please activate the plugin by checking the "Active" checkbox.

Customer-Id Your customer id you received from Wirecard CEE.

Shop-Id

Unique identifier for your shop. (This parameter has to be enabled by Wirecard CEE for your customer id.)

Secret

Your secret you also received from Wirecard CEE.

Service Url

URL to the imprint of your shop, this field is mandatory.

Restore basket

If enabled the original content of the shopping basket will be restored if your consumer did change the basket during the checkout process. If disabled the order will not be executed and set to a clarification state.

Confirm Mail

E-mail address of the merchant for receiving payment information details regarding the orders of the consumers of your web shop. (This parameter has to be enabled by Wirecard CEE for your customer id.)

Show CreditCard Issue-number

If enabled your web shop displays the input field for the issue number within in your credit card form in your web shop. Please be aware that not every card has a issue number.

Use Auto-Deposit

Activating or deactivating the automated deposit immediately after a successful payment of your consumer. (This parameter has to be enabled by Wirecard CEE for your customer id.)

Max. retries

Maximum number of retries during payment

Send additional data

Sending additional data regarding consumer data (consumerBilling data and consumerShipping data)

Save Wirecard response

The response of the Wirecard Checkout contains detailed information regarding the transaction. You are able to store these data in an internal field, which is not visible to your consumer.

Logging

For debugging purposes you can enable the logging. The log data is written to log files within the “/log/” folder of this plug in. Please ensure to set proper access rights for this folder. Please use this logging option only within testing and never within production.

Delete log-file after x days

Count of days until log files are deleted.

Activation of payment types

Please select "Configuration" → "Shipping". Then select on type of shipping in the window "Shipping costs management" for editing. A new window named "Shipping costs" occurs. Please select the tab "Payments". Move your required payment types from the left list named "Available" to the right list named "Selected" by selecting the payment type and pressing the "→" button. After selection is done, please click on button "Save" and close this dialog.

Please select from the main menu "Configuration" → "Payment methods" to activate the new payment types. This is done by selecting one payment type in the left list named "Available payments" and check the "Active" checkbox in the tab "General".

Optionally you have the possibility to modify the description of the payment type. This text is visible to your consumer of the web shop within the checkout process. Additionally you are able to set surcharges and discounts per payment type. Within the "Position" field you are able to change the sort order of the payment types presented within the checkout process in your web shop. After configuring the selected payment type, please click on button "Save" and proceed with the next payment type.
