Information on the protection of personal data for Wirecard Checkout Terminal

Overview

This chapter provides information on all data that are processed by our solution Wirecard Checkout Termianl (WCT).

Wirecard Checkout Terminal data processing volume increases with the services contracted. When ordering Wirecard Checkout Terminal, you may acquire

  • 1 basic solution, optionally extendable with

  • 1 to N payment methods, and/or

  • 0 to N optional and additional features

Go to section “Data entered by the merchant” for an overview on the processing of data (parameters) for a given payment transaction and the selected payment method. Furthermore, section Additional features provides additional information on the processing of data (parameters) for each feature that was added to the basic solution.

Additionally, the general functioning of the product WCT - both in its basic version and when using different payment methods - is also described in detail here: Using WCT
