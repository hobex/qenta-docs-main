Change log

Change log

for Gambio and Wirecard Checkout Seamless

Version 1.2.7 (2018-05-22)

  • Improve style for payolution terms.

Tested versions: Gambio GX3 3.9.1.2 with PHP 5.6, PHP 7.0 and PHP 7.1

Version 1.2.6 (2018-05-09)

  • Add PHP 7.1 compatibility.

  • Add Gambio 3.9 compatibility.

  • Force sending date of birth for payment methods Invoice and Installment.

Tested versions: Gambio 3.7.3.0 with PHP 5.6, PHP 7.0, and PHP 7.0.1

Version 1.2.5 (2018-01-25)

  • Ratepay script only when RatePAY is enabled as payment provider for INVOICE and/or INSTALLMENT.

  • Prevent possible issues with payolution terms when RatePAY is enabled.

Version 1.2.4 (2017-12-20)

  • Add compatibility for Gambio 3.7.2.0.

  • Prevent possible issues with payment method Sofort.

Version 1.2.3 (2017-09-20)

  • Change logo and rename payment method SOFORT to Sofort.

  • Remove the Wirecard text for each payment type.

Version 1.2.2 (2017-08-24)

  • Update payment method logos.

  • Remove deprecated payment methods.

  • Update risk management rules for payment methods Invoice and Installment.

  • Update basket parameters.

  • Add new parameter consumerDeviceId.

Version 1.2.1 (2017-04-18)

  • Revert shopping basket files.

Tested versions: Gambio 3.4.0.0 with PHP 5.6

Version 1.2.0 (2017-01-10)

  • Add Gambio 3.2 compatibility.

  • Update redirect for order state pending.

Tested versions: Gambio GX 3.2.0.1 with PHP 5.6

Version 1.1.0 (2016-11-28)

  • Add new parameter for payment method paysafecard.

  • Improve security on payment server response.

Tested versions: Gambio GX2 2.7.3.1 with PHP 5.6

Version 1.0.3 (2016-08-08)

  • Compatibility with GX3.

Version 1.0.3 (2016-05-18)

  • Support of Honeygrid added.

  • Support of MobileCandy added.

  • Improvements regarding the autoDeposit settings.

  • Improvements regarding the consumer birthday validation.

Version 1.0.2 (2016-02-25)

  • Additional installation step (step 4) added in Installation Guide.

  • Adaptations regarding the new design.

  • Support only for Gambio 2.6 and higher.

Version 1.0.1 (2015-10-28)

  • Improvements regarding the computation of basket item tax.

Version 1.0.0 (2015-08-31)

  • Initial version of plugin v1.0.0 available.
