Change log

Wirecard Checkout Seamless for Magento 2

Version 1.0.13 (2018-10-04)

  • Clear the Magento mini cart after successful payment.

Version 1.0.12 (2018-05-16)

  • Prevent issues with agreements assigner on guest checkout.

  • Add Magento 2.2.4 support.

Version 1.0.11 (2017-11-24)

  • Prevent deleting of an order when payment was not successful.

Version 1.0.10 (2017-11-20)

  • Prevent terms and conditions validation issues during checkout.

  • Remove payment method @Quick.

Version 1.0.9 (2017-11-03)

  • Add Magento 2.2 support.

Version 1.0.8 (2017-10-23)

  • Add checkout agreements module dependency.

Tested versions: Magento Community Edition 2.1.8 with PHP 5.6 and PHP 7.0

Version 1.0.7 (2017-10-12)

  • Rename payment method SOFORT Banking to Sofort./Online bank transfer.

  • Change “SOFORT Banking” logo to Klarna logo.

Tested versions: Magento Community Edition 2.1.8 with PHP 5.6 and PHP 7.0

Version 1.0.6 (2017-09-19)

  • Remove automatic Magento caching in return.

  • Decode error messages.

  • Add specific errors to confirm response.

  • Remove additional information after failed payment.

  • Add PHP 7.1 compatibility (not supported by Magento 2).

  • Update capturing for automated deposit.

Tested versions: Magento Community Edition 2.1.8 with PHP 5.6 and PHP 7.0

Version 1.0.5 (2017-08-16)

  • Update payment method logos.

  • Remove payment methods mpass and Skrill Direct.

  • Rename payment method Bancontact/Mister Cash to Bancontact.

  • Add new parameter consumerDeviceId.

Tested versions: Magento Community Edition 2.1.4 and 2.1.8 with PHP 7.0

Version 1.0.4 (2017-07-10)

  • Remove Magento CVV-validation.

Tested versions: Magento 2 Community Edition 2.1.4 with PHP 5.5 and PHP 7.0

Version 1.0.3 (2017-05-29)

  • Update financial institutions for payment methods eps-Überweisung and iDEAL.

  • Update price calculation method.

  • Prevent possible issues during checkout using payment methods eps-Überweisung and iDEAL.

Tested versions: Magento 2 Community Edition 2.1.4 with PHP 7.0

Version 1.0.2 (2017-04-18)

  • Remove duplicateRequestCheck parameter.

  • IP resolution for systems where proxy is not transparent to application.

  • Allow alphanumeric characters in orderReference parameter and trim to last 10 characters.

Tested versions: Magento 2 Community Edition 2.1.4 with PHP 5.6 and PHP 7.0

Version 1.0.0 (2017-02-14)

  • Initial version of shop plugin.

Tested versions: Magento 2 Community Edition 2.1.4 with PHP 5.6 and PHP 7.0
