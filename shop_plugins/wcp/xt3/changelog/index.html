Change log

Change log

for xt:Commerce 3 and Wirecard Checkout Page

Version 1.11.4 (2018-03-15)

  • Calculate device fingerprint only when Ratepay is enabled for invoice and installment.

Version 1.11.3 (2017-11-16)

  • Remove payment method @Quick.

  • Prevent possible issues when sending confirmation e-mails.

Version 1.11.2 (2017-09-29)

  • Rename payment method SOFORT Banking to Sofort./Online bank transfer.

  • Change "SOFORT Banking" logo to Klarna logo.

Version 1.11.1 (2017-08-29)

  • Add new parameter consumerDeviceId.

Version 1.11.0 (2017-07-31)

  • Implement financial institutions.

  • Implement payment method Masterpass.

  • Implement payment methods Invoice:RatePAY and Invoice by Wirecard.

  • Implement payment method Installment:RatePAY.

  • Update payment methods.

  • Update shopping basket.

  • Update pre-checks for payment methods Invoice and Installment.

  • Update payment method logos.

  • Redesign frontend checkout.

Version 1.10.0 (2016-12-19)

  • Add new parameter for payment method paysafecard.

  • Improve security on payment server response.

  • Update behavior for invalid fingerprint.

Tested version: xt:Commerce 3.0.4 SP 2.1 with PHP 5.3

Version 1.9.0 (2015-09-29)

  • Payment methods Trustly and TrustPay added.

  • Payment method SEPA Direct Debit updated.

  • Payment method Click2Pay removed.

  • Config export formular added.

  • Optional request parameters customerStatement, duplicateRequestCheck, autoDeposit, confirmMail, maxRetries, and paymenttypeSortOrder added.

  • New option to send consumer additional information.

  • New option to send shopping basket information.

  • Device detection integrated.

Version 1.8.0 (2014-09-24)

  • Request parameter customerStatement added.

  • Payment methods ePay.bg, TatraPay and Voucher added.

  • Mobile invoicing renamed to paybox.

Version 1.7.1 (2014-02-05)

  • Support regarding SEPA added.

  • Plugin updated to xt:Commerce 3.0.4.

  • All relevant response data now stored and visible in the shop administration in table wirecard_checkout_page_transaction.

Version 1.7.0 (2013-10-31)

  • New payment methods mpass, Skrill Digital Wallet and Skrill Direct added.
