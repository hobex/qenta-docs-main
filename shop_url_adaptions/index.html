Plugin Shop URL Adaptions

Plugin Shop URL Adaptions

Merchants which are using our shop plugins have to switch the endpoint from Wirecard to new QENTA API.

Old endpoint: checkout.wirecard.com

New endpoint: api.qenta.com

Please change only the first part of the URL. Everything after .com/ stays the same!

Example:

Switch from: https://checkout.wirecard.com/page/init.php or https://checkout.wirecard.com/seamless/frontend/init.php

Switch to: https://api.qenta.com/page/init.php or https://api.qenta.com/seamless/frontend/init.php

Please choose your shop system in the list below and change the endpoint accordingly.
