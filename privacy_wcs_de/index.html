Datenschutzinfo für QENTA Checkout Seamless

Datenschutzinfo für QENTA Checkout Seamless

Überblick

Auf dieser Infoseite können Sie sich über alle Daten informieren, die das Produkt "QENTA Checkout Seamless" (WCS) verarbeitet.

Bei der Bestellung des Produkts WCS kann unterschieden werden zwischen

  • 1 Basisprodukt, optional erweitert um

  • 1 bis N Zahlungsmitteln, und/oder

  • 0 bis N optionalen Zusatzfeatures

Unter Auswahl eines Zahlungsmittels können Sie daher einerseits in der Rubrik privacy_wcs.adoc#Basisprodukt simulieren, welche Daten (Parameter) für eine Zahlungstransaktion mit dem gewählten Zahlungsmittel verarbeitet werden. Andererseits ist in der Rubrik privacy_wcs.adoc#Zusatzfeatures für jedes zusätzlich bestellte Feature ersichtlich, welche Daten (Parameter) addiditiv zum Basisprodukt verarbeitet werden.

Sollten Sie darüber hinaus noch Fragen zu datenschutzrechtlichen Details der einzelnen Parameter haben (zB betroffene Person, Datenkategorie etc), können Sie diesbezügliche Informationen unter privacy.adoc#DatenschutzhinweiseZuEinzelnenParametern einsehen.

Basisprodukt

Datenverarbeitung je Zahlungsmittel

Unter Auswahl eines Zahlungsmittels können Sie hier simulieren, welche Daten (Parameter) für eine Zahlungstransaktion mit einem bestimmten Zahlungsmittel verarbeitet werden. Angezeigt werden dabei sowohl Daten,

  • die vom Merchant oder dem Endkunden an die QENTA Zahlungsschnittstelle gesendet werden (sog. Input-Daten)

  • die von QENTA wieder an diese zurück gehen (sog. Output-Daten).

Sämtliche hier angegebenen Parameter müssen verarbeitet werden, um das Produkt WCS in seiner Basisversion dem Merchant zur Verfügung stellen zu können (Zweck der Datenverarbeitung).

Die hier angezeigten Parameter unterteilen sich in Parameter

  • die unabhängig vom Zahlungsmittel verarbeitet werden (Standard)

  • die abhängig vom jeweiligen Zahlungsmittel verarbeitet werden (Zahlungsmittel)

  • die vom Endkunden eingegeben werden (Interaktion Endkunde)
