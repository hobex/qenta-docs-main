Installation guide for Wirecard Checkout Seamless shop plugin

Installation guide for Wirecard Checkout Seamless shop plugin

This installation guide shows a step-by-step installation of the plugin to your installed shop system on your web server. Please test your web shop and the configuration of the plugin on a test system before installing the plugin on your production system.

Step 1

Copy the extracted files and folders inside "copy_this" directory into the root directory of your OXID shop.

Step 2

If shopversion is >= 4.6.0 just install the extension in shop-admin, for versions < 4.6.0 please insert the module entries below in module configuration:

"Master Settings" → "Core Settings" → "System" → "Modules"

order => dd_wirecard/views/dd_wirecard_order
payment => dd_wirecard/views/dd_wirecard_payment
thankyou => dd_wirecard/views/dd_wirecard_thankyou
oxorder => dd_wirecard/core/dd_wirecard_oxorder
oxpaymentgateway => dd_wirecard/core/dd_wirecard_oxpaymentgateway
oxpaymentlist => dd_wirecard/core/dd_wirecard_oxpaymentlist
oxemail => dd_wirecard/core/dd_wirecard_oxemail

Step 3

Now add the payment methods using the following query (choose the payment methods you want to offer in your shop) via Service → Tools → Update SQL:

  `oxid` varchar(32) NOT NULL,
  `oxshopid` varchar( 32 ) NOT NULL default 'oxbaseshop',
  `oxuserid` varchar(32) default '',
  `sid` varchar(32),
  `insertdate` datetime,
  `status` varchar(32) NOT NULL,
  `classname` varchar(255),
  `functionname` varchar(255),
  `message` text,
  KEY `oxid` (`oxid`),
  KEY `oxshopid` (`oxshopid`),
  KEY `oxuserid` (`oxuserid`),
  KEY `sid` (`sid`),
  KEY `status` (`status`)
);

# --- referencenumber table
CREATE TABLE IF NOT EXISTS dd_orderrefnr (
  `oxid` varchar(32) NOT NULL,
  `oxshopid` varchar( 32 ) NOT NULL default 'oxbaseshop',
  `dd_refnr` int(11) NOT NULL auto_increment,
  `oxordernr` int(11) NOT NULL  default '0',
  `oxuserid` varchar(32) default '',
  `oxusername` varchar(255) NOT NULL default '',
  `oxfname` varchar(255) NOT NULL default '',
  `oxlname` varchar(255) NOT NULL default '',
  `oxtotalordersum` double NOT NULL default '0',
  `oxordercurrency` varchar(3) NOT NULL default '',
  `oxpaymenttype` char(32) character set utf8 collate utf8_general_ci NOT NULL default '',
  `insertdate` datetime NOT NULL default '0000-00-00 00:00:00',
  `status` varchar(32) NOT NULL,
  `sid` varchar(32),
  PRIMARY KEY `oxid` (`oxid`),
  KEY `oxshopid` (`oxshopid`),
  KEY `dd_refnr` (`dd_refnr`),
  KEY `oxpaymenttype` (`oxpaymenttype`),
  KEY `oxusername` (`oxusername`),
  KEY `oxfname` (`oxfname`),
  KEY `oxlname` (`oxlname`),
  KEY `oxuserid` (`oxuserid`)
) ENGINE=MyISAM AUTO_INCREMENT=100000 DEFAULT CHARSET=utf8;
